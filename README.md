# House-Price-Prediction
An end-to-end Machine Learning project that predicts house prices using regression models. Includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and price prediction using Python and Scikit-learn.

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## How to Run
1. Open the notebook.
2. Install the required libraries.
3. Run all cells.
